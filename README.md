■このゲームについて
迷路を自ら作り、自ら迷うゲームです。

■起動方法
「MakeMaze.exe」を実行してください。

■基本操作(デフォルト)
[キーボード]
wasd:移動
esc:マウスカーソル表示、ゲームの説明表示
[マウス]
右クリック:アイテム取得
左クリック:道を開く
マウス移動:視点移動
